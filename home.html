What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents and allows interaction with the structure of a web page as a tree of objects.

How do you select an element from the DOM?

You can select an element from the DOM using methods like getElementById(), querySelector(), or getElementsByClassName().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events for its child elements, improving performance and simplifying code

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using setAttribute() and getAttribute(), and modify its styles using the style property in the DOM.
